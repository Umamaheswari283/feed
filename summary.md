# Dataset Summary by Industry

## Space & Research
- NASA Image Feeds - Astronomy photography collections by Roberto Colombari
- NISAR Satellite Data - L2 GCOV covariance analysis and statistical summaries from Earth observation missions

## Environmental & Sustainability
- USEEIO Environmental Data - Economic input-output sectors, flows, and D-matrix for environmental impact analysis (GAEEIOv1.0-s-20 model)
- Environmental Product Declarations - Product lifecycle environmental impact data
- Land Cover Data - Forest coverage fractions for California counties via Google Data Commons

## Food & Agriculture
- Food Nutrition Database - USDA FoodData Central API providing nutrient information for food items
- Nutrition Labels - Detailed product nutrition data (e.g., cheddar cheese)
- Farm Fresh Markets - Michigan farmers market locations

## Civic Technology & Community
- SeeClickFix 311 Reports - Citizen-reported civic issues (illegal dumping, infrastructure problems)
- Democracy Lab Projects - Civic tech project directory
- Civic Tech Index - Organizations and resource links for civic technology initiatives
- Model Earth Team - Participant tracking for coding meetup integration

## Media & Content
- Film Scouting Locations - Georgia camera-ready filming sites
- BlueSky Social Feed - Model.Earth social media posts via RSS
- Video Collections - Test feeds including BigBunny and linked video content
- GitHub Repository Images - Media assets from Model.Earth repos

## Data Infrastructure & Tools
- Data Pipeline - Python workflow analysis for n8n automation
- RealityStream Parameters - ML feature and target configurations with YAML parameters
- DiffBot API - Knowledge graph query endpoints

## Geographic & Census Data
- US Cities Database - Latitude, longitude, elevation for US municipalities
- Georgia Cities - County-level population data
- US State Populations - Latest census figures via Data Commons
- Data USA - State/territory population statistics

## Manufacturing & Products
- Made in India Products - BuildingTransparency.org product database
- Deep-Containment Architecture - Underground Small Modular Reactor location scouting
